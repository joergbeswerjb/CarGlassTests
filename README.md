# CarGlass Kazakhstan — Assessment Platform

Платформа оценки кандидатов для CarGlass Kazakhstan (франчайзи Belron).

## Технологический стек

- React 18 + Vite 5
- react-router-dom v6
- Google Sheets (Apps Script) как база данных
- Vercel (продакшн)

## Структура

- `/test/:roleSlug` — тест кандидата
- `/hr` — HR-панель (пароль защищён)
- `/done` — экран завершения

## Роли в работе

- ✅ Техник филиала — задеплоен
- 🔜 Operations Manager — следующий
- 📋 GM, CFO, Sales, Lawyer, Finance, Call Centre, IT, Администратор
